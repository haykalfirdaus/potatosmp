CARA BERGABUNG

📱 MOBILE (BEDROCK)

  1. Buka Minecraft > Play > Servers
  2. Klik Add Server
  3. Isi IP & Port di bawah
  4. Klik Save/Play

💻 PC (JAVA)

  1. Multiplayer > Add Server
  2. Masukkan IP di bawah
  3. Klik Done
  4. Join Server
